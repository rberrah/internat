<!-- step:title="Pourquoi ce chapitre" -->
La **polyarthrite rhumatoïde (PR)** et le **lupus érythémateux systémique (LES)** sont deux maladies auto-immunes **systémiques** (non spécifiques d'organe), touchant surtout la **femme**. Elles se distinguent par leur atteinte articulaire et leur profil d'**auto-anticorps**.

:::key
**PR** : polyarthrite **érosive/destructrice** ; auto-Ac = **facteur rhumatoïde** et **anti-CCP**. **LES** : polyarthrite **non érosive** + atteintes viscérales ; auto-Ac = **anti-ADN natif**, anti-Sm, anti-nucléaires.
:::
<!-- /step -->

<!-- step:title="Polyarthrite rhumatoïde : clinique et biologie" -->
Rhumatisme inflammatoire chronique le plus fréquent (~1 %), femme de **40–50 ans**. Polyarthrite **bilatérale, symétrique**, touchant **poignets, MCP, IPP** (respect des IPD), douleurs **inflammatoires** (réveil nocturne, **dérouillage matinal > 30 min**), synovites, nodules rhumatoïdes.

:::key
Biologie : **syndrome inflammatoire** (VS, CRP, hyperleucocytose à PNN, thrombocytose, anémie inflammatoire). **Facteur rhumatoïde** (auto-Ac IgM anti-IgG, néphélémétrie/ELISA, Se 60–80 %, Sp 65–85 %, **non spécifique**). **Anti-CCP/ACPA** (ELISA, Se 60–75 %, **Sp 90–95 %**), **spécifiques** et **pronostiques** (maladie érosive).
:::

Diagnostic : critères **ACR/EULAR** ; imagerie = **érosions**, pincement articulaire.
<!-- /step -->

<!-- step:title="Traitement de la PR" -->
:::clinical
Le traitement de fond est une **urgence thérapeutique** (« fenêtre d'opportunité »). **Méthotrexate** = traitement de fond de **première intention** (surveillance NFS, bilan hépatique, rénal ; CI : grossesse, insuffisance hépatique/rénale). En cas d'échec : **biothérapies** — **anti-TNF**, **anti-IL6R** (tocilizumab), **CTLA4-Ig** (abatacept), **anti-CD20** (rituximab, 3ᵉ ligne) — et **inhibiteurs de JAK**, en association au MTX. Corticoïdes en attendant l'effet du traitement de fond.
:::

:::note
Avant une biothérapie : rechercher une **tuberculose latente** (IDR/IGRA, radio pulmonaire), les sérologies virales et un foyer infectieux (risque de réactivation).
:::
<!-- /step -->

<!-- step:title="Lupus : clinique et biologie" -->
Maladie multifactorielle, **9 femmes / 1 homme**, en période d'activité génitale. Atteinte articulaire (polyarthrite **non déformante, non érosive**), **cutanée** (rash malaire en vespertilio, **photosensibilité**), **rénale** (glomérulonéphrite lupique, **pronostique**), hématologique, neurologique.

:::key
Biologie : **dissociation VS accélérée / CRP normale** (sauf sérite ou infection), **cytopénies auto-immunes** (anémie, lymphopénie, thrombopénie). Auto-Ac : **anti-nucléaires (AAN/FAN)** par IFI sur cellules Hep2 (**> 99 %**, dépistage, peu spécifiques) ; **anti-ADN natif** (Farr, Crithidia, ELISA — **spécifique**, corrélé à l'atteinte rénale, marqueur de **suivi**) ; **anti-Sm** (très spécifique mais rare), anti-SSA/SSB/RNP, **anti-histones** (évocateurs d'un **lupus induit médicamenteux**) ; **hypocomplémentémie** (CH50, C3, C4 abaissés lors des poussées).
:::
<!-- /step -->

<!-- step:title="Traitement du lupus" -->
:::clinical
Traitement de fond : **hydroxychloroquine** (Plaquenil) — réduit les poussées ; principale toxicité **rétinienne** dose-dépendante → **surveillance ophtalmologique**. **Corticoïdes** lors des poussées. **Immunosuppresseurs** (cyclophosphamide, mycophénolate mofétil) pour les formes viscérales/rénales graves. **Bélimumab** (anti-BAFF/Blys), rituximab (hors AMM) dans les formes réfractaires.
:::

:::note
Rechercher un **syndrome des anti-phospholipides (SAPL)** associé (anticoagulant lupique, anti-cardiolipine, anti-β2GP1) : risque **thrombotique** et obstétrical. Grossesse à **programmer** (arrêt des tératogènes : MMF, cyclophosphamide, méthotrexate).
:::
<!-- /step -->

<!-- step:title="PR ou lupus : les clés" -->
:::key
- **Articulation** : PR = **érosive/destructrice** ; LES = **non érosive, non déformante**.
- **Inflammation** : PR = VS **et** CRP élevées ; LES = **VS ↑ / CRP normale**.
- **Auto-Ac** : PR = FR + **anti-CCP** ; LES = AAN + **anti-ADN natif** + anti-Sm.
- **Fond** : PR = **méthotrexate** ; LES = **hydroxychloroquine**.
:::
<!-- /step -->

<!-- step:title="A retenir" -->
- Deux maladies auto-immunes **systémiques**, prédominance féminine.
- **PR** : polyarthrite érosive symétrique ; **anti-CCP** spécifiques ; fond = **MTX** puis biothérapies.
- **LES** : atteinte multiviscérale (rénale pronostique) ; **anti-ADN natif**/anti-Sm spécifiques ; **hypocomplémentémie** en poussée ; fond = **hydroxychloroquine** ± immunosuppresseurs.
- Penser au **SAPL** dans le lupus.
<!-- /step -->
